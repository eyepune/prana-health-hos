# 🌿 Prana Health Intelligence — Health OS Landing

This is the premium, clinical-grade landing page for **Prana Health Intelligence**, a sovereign Health Operating System built for Bharat and the world. 

## 🚀 Key Features

- **The Tri-Intelligence System**:
  - **Symptom Analyst**: Powered by Llama 3.1 8B for clinical-grade guidance.
  - **Report Interpreter**: Powered by Llama 3.2 11B Vision for biomarker tracking (Life-Map™).
  - **Product Lens (Guardian)**: Real-time scan for food and medication safety.
- **Privacy-First Architecture**: Built on Open Source LLMs (Bytez API) to ensure data sovereignty.
- **Sachet Pricing Model**: Optimized for the Indian market (₹49 sachet hooks to ₹499 family plans).
- **Safety First (Red Shield™)**: Proprietary safety layer to prevent contraindications and provide emergency routing.

## 🛠 Tech Stack

- **Frontend**: HTML5, Vanilla CSS3 (Custom Design System), vanilla JavaScript.
- **Design**: Bio-minimalist aesthetic with glassmorphism, dynamic particles, and high-performance animations.
- **Intelligence**: Bytez API (Llama 3.1/3.2).
- **Backend & Auth**: Supabase (PostgreSQL).
- **Deployment**: Optimized for Vercel/Edge.

## 🏃 How to Run Locally

1. Open the directory in your terminal.
2. Run the following command (requires Node.js):
   ```bash
   npx serve .
   ```
3. Open `http://localhost:3000` in your browser.

## 📁 Project Structure

- `index.html`: Main landing page structure with SEO and Accessibility best practices.
- `style.css`: Comprehensive design system, typography, and responsive layouts.
- `main.js`: Interactive components, particle background, scroll-reveal animations, and waitlist handling.
- `package.json`: Project configuration.
